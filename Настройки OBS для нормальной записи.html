Настройки OBS для нормальной записи видио с экрана

OBS-studio — бесплатная программа с открытым исходным кодом для записи видео и потокового вещания.
Так, например, введя «Имя компонента не содержит раскос», вы даете Tekla Structures указание отобразить все компоненты, имена которых не содержат слова «раскос». Соответственно, Tekla Structures скрывает все компоненты, в именах которых слово «раскос» присутствует.

Сайт программы: https://obsproject.com/ru

Загрузить портативную версию программы в zip архиве по ссылке: https://obsproject.com/download

Настройки вывода

Параметры влияющие на объём конечного файла:
Качество записи — Высокое качество, средний размер файла
Кодировщик — Аппаратный (QSV, H.264)

Формат записи: mkv или flv. т.к. при краше программы видеозапись с этим расширением будет сохранена и спасена. Файлы с остальными расширениями будут безвозвратно потеряны.
